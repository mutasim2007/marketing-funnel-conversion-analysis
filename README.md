# Marketing Funnel & Conversion Performance Analysis

## 📊 Project Overview
This project focuses on analyzing a marketing funnel to understand conversion performance, identify drop-offs, and evaluate channel effectiveness.

The goal is to optimize the lead-to-customer journey and improve overall conversion rates using data-driven insights.

---

## 🎯 Objectives
- Analyze funnel stages and conversion rates  
- Identify major drop-off points  
- Evaluate performance by lead source  
- Understand user behavior across visits  
- Provide actionable recommendations  

---

## 🛠 Tools Used
- Power BI (Dashboard & Visualization)  
- Python (Data Analysis)  
- Excel (Data Preparation)  

---

## 📈 Key Insights
- Significant drop-off observed between initial visits and conversions  
- Higher conversion rates from high-intent leads  
- Lead sources like referrals perform better than some paid channels  
- Users with more visits tend to have higher conversion probability
